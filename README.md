# Repetition-Water-Tower-Program

## Description
- A C#/.NET program that calculates the most efficient distance between 3 separate houses to install a water tower between them.
- 
## Technologies Used
- C#
- .NET
- Visual Studio
  
## Features
- Calculating distances on a 15x15 scale ratio.
- User input validation.
